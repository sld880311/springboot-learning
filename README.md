#### setting配置文件说明
     为了更快的下载第三方依赖包，可以配置国内的仓库，配置参考如下：
     	
     
      < !-- 在mirrors 中配置 mirror -->
      <mirror>
        <id>alimaven</id>
        <name>aliyun maven</name>
        <url>
            http://maven.aliyun.com/nexus/content/groups/public/
        </url>
        <mirrorOf>central</mirrorOf>        
      </mirror>
